PS4 Key Serialization for Diva Pico

Pack signature.bin, serial.txt, and key.pem into one copy-friendly plain text string for the new ps4key CLI command. The format is PS4K1 + Base64(binary payload) with a CRC32 over the raw payload bytes.

Input

Provide the binary signature, the 16-digit serial, and the RSA private key text. Validation is intentionally strict to catch common mistakes before flashing.

Output

Single-line plain text, suitable for direct paste into the CLI as ps4key <serialized_key>.
